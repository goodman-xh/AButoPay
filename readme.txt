AButoPay adds a TRC-20 USDT payment gateway to WooCommerce with automatic monitoring and subscription management.

== Description ==
AButoPay provides a complete USDT-TRC20 payment solution for WooCommerce:
* Auto-generate and monitor payment addresses  
* Display partial/full payment status with retry logic  
* Manage subscription plans and free payment counts  
* Admin area QR code and copy-to-clipboard  

== Installation ==
1. Upload the `abutopay` folder to `/wp-content/plugins/`  
2. Activate “AButoPay” in the Plugins screen  
3. Go to WooCommerce → Settings → Payments → AButoPay to configure API keys  

== Frequently Asked Questions ==
= How do I enable automatic direct messaging? =
Go to the AButoPay settings page, enable “Auto DM” and enter your message template.

= How do I renew my subscription? =
In the “Upgrade Plan” tab, pay the generated USDT address or scan the QR code.
